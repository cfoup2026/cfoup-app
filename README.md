# CFOup App

CFO inteligente para pequenas empresas.

## Pergunta central
Quanto dinheiro vou ter daqui a 90 dias?

## Posicionamento do produto
CFOup não é software de contabilidade.
É uma camada de inteligência financeira construída em cima dos dados que a empresa já possui, para ajudar o dono a tomar melhores decisões de caixa.

## Estágio atual
MVP piloto em execução para um caso de uso de pequena empresa no Brasil.

Este repositório é a base para definição do produto, execução do MVP e construção da aplicação.

## Foco do MVP
- onboarding

- alertas básicos

## Fora do escopo do MVP

# CFOup App

AI CFO for small businesses.

## Core question
How much cash will I have in the next 90 days?

## Product positioning
CFOup is not accounting software.
It is a financial intelligence layer built on top of existing business data to help owners make better cash decisions.

## Current stage
Pilot MVP in execution for a small business use case in Brazil.

## Repository purpose
This repository is the base for product definition, MVP execution, and application build.

## Main documents
- `CFOup_MasterPRD_v1.1.docx` → master product document, vision, MVP boundaries, and post-MVP roadmap
- `CFOup_MVP_Execution_PRD.md` → execution PRD for MVP development and delivery

## MVP focus
- onboarding
- bank statement upload
- transaction classification
- cash dashboard
- 90-day forecast
- basic alerts

## Out of scope for MVP
- AI chat assistant
- Stripe billing
- bank API integration
- multi-company support
- mobile app

## Stack
- Next.js
- Supabase
- Claude
